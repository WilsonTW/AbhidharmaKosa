# AbhidharmaKosa

由[世親菩薩](https://zh.wikipedia.org/wiki/%E4%B8%96%E4%BA%B2)所著，部派佛教教理之集大成，大毗婆沙論之綱要書。略稱俱舍論。意譯對法藏論、聰明論。

## 分別界品

**界品**

-   總說一切法體(五位七十五法)
-   有漏無漏法，除道餘有為

**一切法(分類一)**

-   有漏
	-   苦
	-   集
-   無漏
	-   滅
	-   道

**一切法(分類二)**

-   有為
	-   苦
	-   集
	-   道
-   無為
	-   滅

**一切法(分類三)**

-   有為法
	-   色法(11)
	-   心法(1)
	-   心所法(46)
	-   不相應行法(14)
-   無為法(3)

**法(Dharma)：體用**

1.  原理，規則(Principle) => 緣起
2.  原素(Elements) => 根本
3.  教法(Teaching) => 佛陀/學者所闡述
4.  一切(All) => 現象
5.  涅槃(Nirvana) => 親證

**色法(5根 + 5境 + 無表色)**

**極微論(極微 => 7極微 = 1微團)**

**四大種**

-   地(堅：能持)
-   水(濕：能攝)
-   火(煖：能熟)
-   風(動：能長)

**根 (Indrya)：最勝, 增上, 光顯**

 - 勝用：取境(五境)之作用
 - 增上：發識(五識)之作用
 - 光顯：有能見聞之光明(體)

**境：色聲香味觸**

**無表色**

1.  “表”由身口而表現令之(五根能取)
2.  “無表”不能見,  聞,  覺,  知
3.  定心位或發自內心的善惡,  由身口表現(有表)後,  所留下的力用能持續一段時間名為”無表色”
4.  體性：

	-   體：體非極微,  由四大種所造,  由身口表所引發,  故名色
	-   性：唯善惡二性

6.  作用：

	-   造就習性
	-   有妨善妨惡之功能(ex. 受戒之善體)

**心法**

1.  一切的心理和精神作用稱”心法”
2.  心王：認識的主體, 一切心理作用的源頭
3.  心所：各種不同的心理作用
4.  心王取總相, 心所取別相

	-   總相：ex. 青色, 車
	-   別相：ex. 色之深淺, 什麼車

**心法(法數)**

1.  心王：六識合為一心王
2.  心所法：

	1.  大地法(10)
	2.  大善地法(10)
	3.  大煩惱地法(6)
	4.  大不善地法(2)
	5.  小煩惱地法(10)
	6.  不定地法(8)

**心心所相應(五種平等)**

1.  所依平等：所依”根”
2.  所緣平等：所緣”境”
3.  行相平等：境映心的相
4.  時平等：同時起
5.  事平等：事是”體”, 心王, 心所各有其不同之體及用, ex. 苦, 樂二受不會同時一念中起

**心王：心, 意, 識**  (p.s. 唯識之六七八識)

1.  心(citta)：集起．總稱：能引起諸心心所及三業所造種種事業
2.  意(manas/末那)：思量：思量所緣境及為心心所生起之依止
3.  識(vijnana)：了別．認識：分別不同之境相而產生認識

**心**

1.  Citta：集起, 種種：mind, member 總稱：一切精神體之統一
2.  集起：一切心心所法(種種)所集起．由造作, 認識留下的一切所集(如資料庫)
3.  由心力引起諸心所並由之造作種種事業(主導的動力)
4.  唯識稱第八識為”心”

**意**

1.  Manas(末那)：思量 intellect, intelligence, understanding, perception, sense
2.  思量所緣境
3.  所依：心心所生起之依止

	-   身心交感中樞：為五根之所依, 五根依意根才能存在或產生作用
	-   認識作用的源泉：依意根生前五識乃至意識

**心(五識意識) <--(意根)--五根 <--(緣)--五境**

**意根**

1.  意根：前剎那滅去之”識”
2.  作用：具有引發後剎那之”識”的作用, 稱為無間滅意
3.  ex.

	1.  前剎那滅去之眼識能引發後剎那之眼識
	2.  前剎那滅去之意識能引發後剎那之眼識

	**(心識的作用為剎那生滅相續)**

**識**

1.  vijnana：分別, 了別．the act of distinguishing or discerning
2.  三種分別

	1.  自性分別
	2.  計度分別
	3.  隨念分別

3.  六識合為一心王

**識之三種分別**

1.  自性分別：五識直接將對境之特性接收，而不加任何判斷, 分別, 又稱”無分別”
2.  計度分別：對資訊的分析, 判斷, 概念組構等, 如這是黃色, 花
3.  隨念分別：記憶, 反思等等, 如這是母親最喜歡的花等

**心王(六識)**

1.  依根及所緣分六識
2.  六識：

	1.  眼識：依眼根, 緣色境, 了別色境
	2.  耳識：依耳根, 緣聲境, 了別聲境
	3.  鼻識：依鼻根, 緣香境, 了別香境
	4.  舌識：依舌根, 緣味境, 了別味境
	5.  身識：依身根, 緣觸境, 了別觸境
	6.  意識：依意根, 緣法境, 了別法境

**別法處**

1.  意識所緣境稱為”法處”, 不同於其他”法”的定義
2.  所有意識能認識的, 想得到的都在此”法處”的範圍, 一切能思考, 認識的”對像”
3.  包括75法中的無表, 心所, 不相應及無為等64法
4.  除五根, 五境外, 五境為五識之特定所緣

**六識依根立名**

1.  所依勝：識功能隨根轉, 識功能不隨境轉
2.  依不共因

	1.  各人有各之五根稱為不共, 但五境則是所有人所共取
	2.  根為主觀能取, 境為客觀所取

**心所法**

1.  隨應心王而起的各種心理作用
2.  分類：46法

	1.  大地法(10)
	2.  大善地法(10)
	3.  大煩惱地法(6)
	4.  大不善地法(2)
	5.  小煩惱地法(10)
	6.  不定地法(8)

**大地法(10)**

<img src="https://github.com/WilsonTW/AbhidharmaKosa/blob/master/res/3.png" width="80%" height="80%"/>

1.  大：週遍
2.  地：所行處之心王, 心起處
3.  法：心所法
4.  遍一切心王所行處, 只要六識任一起, 大地法必共起
5.  **作意**, **觸**, **受**, **想**, **思**, **欲**, **勝解**, **念**, **三摩地**, **慧**

**作意**

1.  原文：能令心警覺
2.  Manaskara：consciousness, attention of the mind
3.  令心起警覺, 專注於所緣境上
4.  根境合和時, 由於心的警動, 才發為認識, 此”警動”即為”作意”

**觸**

1.  觸心所：根, 境, 識三和合所生

	 - 經部：三和合成觸
	 - 有部：三和合生觸

**受(想蘊)**

1.  領納為性, 對外境的一種感覺, 情緒的反應作用
2.  五種

	1.  苦受

		-   身苦：前五識
		-   心憂：第六識

	2.  樂受

		-   身樂：前五識
		-   心喜：第六識

	3.  捨受

		-   不苦不樂

**想(想蘊)**

1.  取像：對外境取映於內心而形成一種內心的概念
2.  如：內心的顯形色, 語言之言調

**思(行蘊的體)**

1.  令心造作
2.  有情的意志作用
3.  對外境的意志反應
4.  能以善, 惡, 無記推動其他心心所去造作
5.  意業之體, 身口業之依而起

	-   思：意業
	-   思所做：身口業

**欲**

1.  原文：希求所作事業
2.  釋：對境界希求, 想要什麼, 想做什麼
3.  可善可惡

**慧**

1.  原文：於法能有簡擇
2.  對一切法的是, 非, 善, 惡, 染, 淨的辨別力, 抉擇力是此心所的功能

**勝解(Adhimukti)**

1.  原文：能於境印可
2.  Adhimukti : confidence
3.  對境界堅固不移的殊勝理解(信念), 由深刻的理解後達到內心肯定的作用

**念(Smriti)**

1.  原文：於”緣”明記不忘
2.  Smriti：remembrance, reminiscence, thinking of or upon
3.  對過去的經驗或所緣明記不忘

**三摩地(samadhi) => 定的通稱**

1.  原文：心一境性
2.  Samadhi：等持 putting together, joining
3.  令心專注於一境的作用, 有此心所, 我自才能精神集中
4.  定品詳說

**大善地法：與一切善心俱起, 體惟善性(10)**

<img src="https://github.com/WilsonTW/AbhidharmaKosa/blob/master/res/4.png" width="80%" height="80%"/>

1.  信
2.  不放逸
3.  輕安
4.  捨
5.  慚
6.  愧
7.  無貪
8.  無嗔
9.  不害

**信**

1.  原文：令心澄淨, 於諦實業果終現前忍許(深刻的認識)
2.  faith, believe, trusting, confidence
3.  對事物深刻暸解(認許)之後由無疑惑(心淨)所產生的信念, 特別是只對三寶四諦業果等的信念

**不放逸**

1.  原文：修諸善法離諸不善法, 復何名修, 謂此於善專注為性, 餘部經中有如是釋, 能守護心名不放逸
2.  Apramada：careful, cautious
3.  專注於善法的修行, 將心維持在善法上, 令不忘失

**輕安**

1.  原文：心堪任性
2.  Prasrabdhi：ease
3.  身心的輕便與安適, 身心輕便安適可增加對外的抵抗力和抗壓性, 故說為”堪任性”

	 - 身輕安：與前五識相應
	 - 心輕安：與意識相應

**捨**

1.  本文：心平等性, 無警覺性, 說名為捨
2.  Upekga：indifference, endurance, abandonment
3.  心平等性：用功於善法時, 使心平等, 不浮不沉, 不緩不躁, 平等前進
4.  無警覺性：不輕易受打擾

**慚**

1.  無慚：

	 - 於諸功德及有德者, 無敬無崇無所忌難無所隨屬說名無慚
	 - 於所造罪自觀無恥名曰無慚

2. 與無慚相反者即是慚：

	 - 對善法, 善事即有德者, 能起崇敬, 想要跟隨之心, 能分別善惡, 並進而能起行善之心, 為自己的罪惡或過錯為恥

**愧**

1.  無愧：

	 - 恭敬所敵對法, 為諸善士所厭惡之法說名為罪, 在此罪中不見怖畏說明無愧
	 - 對自己所造之惡, 觀他無恥說名為愧

2. 與無愧相反者即是愧：

**無貪與無嗔**

1.  無貪：

	 - 對順意之人事物的不染著就叫無貪

2.  無嗔：

	 - 對不順意人事物的不憎恨而能夠接受

**#不害**

1.  原文：無損惱
2.  Ahimsa
3.  不脅迫, 不迫害, 不惹惱他人或有情, 是無嗔的粗顯形象, 無嗔可以起慈, 不害可以拔苦叫悲

**勤**

1.  原文：令心勇悍為性
2.  精勤於止惡行善
3.  對不善法的勤勞稱為懈怠, 斷惡修善
4.  四正勤

	1.  未生惡令不生
	2.  已生惡令速斷
	3.  未生善令速生
	4.  已生善令增長

  

**大煩惱地法(6)**

<img src="https://github.com/WilsonTW/AbhidharmaKosa/blob/master/res/5.png" width="80%" height="80%"/>

1.  遍一切不善煩惱心：當一有**不善**和**煩惱**心念起來時, 此六心所必俱起
2.  **煩惱**：對身心的擾亂, 令不自在念稱”煩惱”,
3.  **不**善：對他起惡念謂不善
4.  有六法：**痴, 放逸, 懈怠, 不信, 惛沈, 掉舉**

**痴**

1.  原文：愚癡, 即是無明, 無智, 無顯
2.  愚癡Moho：darkness or delusion of mind, 心的矇晦, 不清
3.  無明Avidya：ignorance, spiritual ignorance, 對事理錯誤的認知
4.  無智Ajnana：unknowledgeable, 對事理無法作正確的判斷
5.  無顯Asamprakhya：invisible, unclear, not bright, 矇晦, 不清

**放逸**

1.  原文：”不修諸善”, 是”修諸善”所對治法
2.  Pramada：negligence, carelessness about
3.  做事不用心, 不精勤

**懈怠**

1.  原文：”不修勇悍”, 是前”勤”所對治法
2.  Kausidya：sloth, indolence
3.  不想做事

**不信**

1.  原文：”心不澄淨”, 是前”信”所對治法

**惛沈**

1.  身心沈重, 身重性, 心重性, 身心無堪任性, 身惛沈, 心惛沉, 故名”惛沈”
2.  Stydna：dense, stiffen, bulky
3.  前五識相應：身惛沈, 第六識相應：心惛沉

**修定：要離五欲(色聲香味觸)五蓋(貪,嗔,惛沈睡眠,惡作掉舉,疑)**

**掉舉**

1.  原文：令心不靜
2.  心浮動性, 心念高舉, 內心不平靜, 打妄想, 胡思亂想
3.  掉舉惡作為修定的障礙, 無法專注於所緣境

	1.  心掉舉
	2.  身口掉舉
	3.  惡作：對過去作為(善,惡)的追悔

**小煩惱地法(10)**

<img src="https://github.com/WilsonTW/AbhidharmaKosa/blob/master/res/7.png" width="80%" height="80%"/>

1.  無明相應,  個別起,  唯第六意識相應,  修所斷

	**無明：**

	 - 見惑：知見 => 見所斷
	 - 思惑：情意 => 修所斷

2. 有十法

	-   **忿，覆，慳，嫉，惱，害，恨，諂，誑，憍**

**忿**

**覆**

1.  隱藏自己過失的心理

**慳**

1.  慳吝, 不肯助人的心理作用, 包括不願意花時間, 財務, 精神, 技巧, 是貪的等流類

**嫉**

1.  嫉妒, 不耐他榮, 對別人的成就心不喜

**惱**

1.  惱害, 不高興, 做錯事不喜人規勸

**害**

1.  為害, 迫害, 脅害, 惹惱他人或有情

**恨**

1.  怨恨, 不滿的情緒一直在心中, 是嗔的等流

**諂**

1.  諂曲, 內心有不同意見不願意表達而表面迎合

**誑**

1.  欺騙他人的心理

**憍**

1.  驕傲, 染著自己之成就自覺了不起(慢：自他相較而心生高)

**不定地法(8)**

<img src="https://github.com/WilsonTW/AbhidharmaKosa/blob/master/res/8.png" width="80%" height="80%"/>

1.  有個別作用：不攝於前五法中
2.  性質不定：有些法可通善, 惡, 無記三性
3.  八種：**尋, 伺, 睡眠, 惡作, 貪, 嗔, 慢, 疑**
4.  可分三類：

	1.  通三性：善, 惡, 無記

		1.  **尋**：內心粗顯的推求作用
		2.  **伺**：內心審細的考察作用
		3.  **睡眠**：為調整身心稱善, 過度睡眠不修善法稱惡

	2.  通善惡：

		1. **惡作**：追悔為性

			 -   善：追悔過去所做之惡行或未做之善事
			 -   惡：追悔過去所做之善行或未做之惡事

		**不論善惡,  過度追悔都為修行(定)之障礙**

	3.  為不善煩惱：4種

		1. **貪**：對順境起染愛

		2. **嗔**：對逆境起厭離

		3. **癡**：對三寶,  四諦之懷疑

		4. **慢**：在自他比較下所生起的高傲之心

			1. 慢：a. 輕視不如自己者 b. 不尊重同等自己者

			2. 過慢：a. 自認勝於跟自己一樣者 b. 不服勝於自己者

			3. 慢過慢：於勝計己勝, 認為自己強於勝過自己很多者

			4. 我慢：以五蘊身心著有我而起的自我中心感

			5. 增上慢：未證自錯認為證：果位及定功德

			6. 卑慢：自卑而產生的傲慢

			7. 邪慢：邪見所產生的傲慢, 如以自作惡事為大功德斷滅論者的無因無果

**不相應行法(14)**

<img src="https://github.com/WilsonTW/AbhidharmaKosa/blob/master/res/9.png" width="80%" height="80%"/>

1.  行蘊所攝(心所法 & 非心所法)其中之**非心所法**部分
2.  非色法(物質)非心法(精神)
3.  有部認為是”實法”, 經部認為是”假法”
4.  解釋：


	 -   因果關係的自然法則,  不離有為生滅
	  -   抽象概念
	 -   為修行而立
	 -   哲學性質

5.  十四種：

**得,**  **非得,**  **命根,**  眾**同分,**  **無想果,**  **無想定,**  **滅盡定,**  **生,**  **住,**  **異,**  **滅,**  **名,**  **句,**  **文**

**得(prapti)**  **＆非得(aprapti)**

1.  得：能使我們”獲得”某事物的”法則”
2.  非得：能使我們”失去”某事物的”法則”
3.  惟發生在自相續和二擇滅

	-   自相續：自己身(生理)心(精神)上
	-   二擇滅：擇滅和非擇滅
	-   表示”修在靠自己”的一種理念

5.  獲：初**得**名獲：a. 未得今獲 b. 已失今獲
6.  成就：**得**後能保持一段時間(ex. 戒體)

**命根**

1.  命根體即壽,  能持煖及識
2.  指壽命,  即生命體,  或維生命的原理
3.  規範我們生命作用能持續運作的原理
4.  由過去業力所引所持
5.  生命現象 - 壽,  煖,  識三相依持

	-   壽：由業力所引之生命體
	-   煖：生命之生理作用
	-   識：生命之精神作用

**眾同分**

1.  讓有情眾生具有共通特點的法則是從因來說
2.  有情眾生具有的共通特點的法則是從相似來講
3.  兩種：

	1.  無差別同分：同種類的共通性質
	2.  有差別同分：不同種類的不同性質, 使一個種類和其他種類能有所區分

**無想果**

1.  無想天之果報, 由修無想定而來
2.  修什麼定, 由生前之訂立餘勢, 死後就會投生於之相應天界中, 餘勢一失就再投胎(欲界)
3.  位置：色界 - 第四禪 - 廣果天 - 高勝處
4.  此為實有物, 能遮未來心心所法令之不起

**無想定**

1.  欲得無想果而修之定
2.  外道(印度教)認為精神作用之止息為解脫
3.  非佛教聖者所修, 非真解脫之方法
4.  止息心心所為先
5.  世親和經部認為是假法, 且細心非完全止滅
6.  有部認為是”實體”, 且完全無心

**滅盡定(NIRODHASAMAPATTI)**

1.  此滅盡定為求靜住, 止息想作意為先．......此滅盡定唯在有頂(天), 即非想非非想處．
2.  靜住：涅槃的寂靜之住．佛教聖者為球相似涅槃之寂靜而修之定
3.  又名滅受想定, 以非想非非想處之有天為依處, 止息受想為先而修
4.  三果聖者能入：見所斷及非想非非想定之能力者方能起
5.  為有漏定,  不與無漏慧觀相應
6.  唯識認為是無漏定而放於無為法中

**生住異滅四相**

1.  四相本為萬物變化的四種現象(經部), 有部將其實體化後, 成為現象背後的法則
2.  經部：生：本無今有, 住：相續隨轉, 異：前後有別, 滅：相續之斷
3.  有部認為是四種現象背後的原理, 而非單是現象, 故為”實法”, 如”生相”是令有為法生起之體
4.  四本相(生,住,異,滅)&四隨相(生生, 住住, 異異, 滅滅)

	-   四本相能作用於物和其他三本相上,  四隨相上
	-   四隨相只能作用各自知本相上
	-   九法俱起,  八一有用(本相對八個有用,  隨相對一個有用)

**名句文**

1.  語言作用本為世間共許下,產生的使用規則, 但這作用不只涉及聲音的高低曲折或圖形的變化, 更影響到我們的認識層面, 有部將這些複雜規則和其所產生的認識實體化後, 便成實有法
2.  也可以說是佛教對印度語言的一種系統分析和規範
3.  語言組構：名(nama), 句(pada), 文(vyanjana)

**三三昧(三解脫門)**

1.  於四禪定(初禪, 二禪, 三禪, 四禪)中起無漏慧觀
2.  無漏慧觀：

	1.  空
	2.  無相
	3.  無願

**意識**

-   五俱意識
-   獨頭意識

**文**

1.  Vyañjana：mark, letter
2.  指單音或字母：如a(阿), ā(阿~)
3.  印度古語或梵語為拼音文字,  如現在西方語言
4.  目前所知的梵文由13母音和35個子音所組成
5.  母音：a ā i ī u ū ṛ ṝ ḷ ḹ e ai o au
6.  子音：ka kha ga ghi na/ca cha ja jha ……
7.  二文以上稱為文身,  即成為名
8.  有部認為是實法

**名**

1.  Nāma：name, term, vocabulary
2.  名稱, 單字
3.  “名謂作想”, “隨音聲歸赴於境”
4.  名稱具有指涉事物, 讓內心產生外物概念(想)的作用
5.  rūap, bodhi, Buddha
6.  有部認為是實法

**句**

1.  pada：locality of name
2.  句者謂章, 詮益究竟
3.  名有系統規則組合而成為句, 如此能表達出完整的意思
4.  gate gate pāragate pārasaṃgate bodhi svāhā
5.  因為句有詮義之功能,  故為實法

**無為法**

1.  Asaṃskṛta-dharma
2.  a+saṃ+skṛ

	1.  saṃ：together
	2.  skṛ：do, act
	3.  saṃskṛta-dharma：put together
	4.  composed, conditioned or active things
	5.  有造作,  因緣所成,  有為法
	6.  a否定 —> 無造作,  無因緣所成,  無為法
	7.  沒有生滅變化

**阿含經：**

1.  理體與現象

	1.  緣生法：萬物生滅變化的現象 —> 有為法
	2.  緣起法：現象背後的法則理本 —> 無為法

2.  輪迴與涅槃

	1.  有為法：生命的現象 —> 惑業苦的無始輪迴
	2.  無為法：涅槃 —> 惑業苦的終止

**說一切有部(三種)：**

1.  有為無為同一層次, 非理事之關係, 故有為無為不必然交涉相關

	-   以修行解脫為中心而成立
	-   **虛空無為**：ākāśā
	-   **擇滅無為**：pratisajkhyā-nirodha
	-   **非擇滅無為**：apratisajkhyā-nirodha

**大眾部：**

1.  理事之關係同阿含經之緣起與緣生

**大乘：**

1.  本體與現象(與理事稍有不同)：無為法是本體, 有為的法是現象, 如水與波, 涅槃為萬法之體 —> 無為, 緣起與緣生皆為現象 —> 有為

**虛空無為** **ākāśā**

1.  定義：虛空無為是空間的特性, 是超越係向生滅背後的法理
2.  其法體是實體有, 因為有這樣的法理一切物質能在空間中間活動, 來去自如
3.  一般我們所謂的空間稱為空界是可見的色

**虛空無為 v.s. 空界**

-   阿含經：虛空=空界
-   上座部：虛空=空界 —> 有為
-   經部：虛空=空界 —> 無為 —> 假法
-   有部：空界 —> 有為法 ; 虛空 —> 無為法：理體

**擇滅無為** **pratisajkhyā-nirodha**

1.  擇滅以離繫為性
2.  擇：簡擇, 為”慧”的作用
3.  滅：煩惱滅, 究竟滅為”涅槃”
4.  阿含經之無為=涅槃
5.  離繫：離煩惱
6.  擇滅無為：以智慧簡擇, 漸滅煩惱達涅槃
7.  智慧簡擇是有為, 煩惱滅是無為

**非擇滅無為** **apratisajkhyā-nirodha**

1.  畢竟礙當生
2.  緣缺不生：由緣缺(礙)而永不生

	-   煩惱之非擇滅：如煩惱斷後成聖人,  不再具凡夫性,  缺乏煩惱再生的因緣,  即為煩惱之非擇滅
	-   一般有為法：眼識生時,  同一剎那之耳識不生即成緣缺永不生

**蘊,  處,  界三科**

1.  佛法以有情為本, 就應該認識有情是什麼, 佛常用”三處觀”去觀察有情, 分別有情的真相
2.  依之來說明”緣起無我”
3.  對有情的觀察分析：

	-   空間上：三種不同面向的分析：蘊, 處, 界
	-   時間上：流轉相續的流變觀：十二緣起

4.  有情是色心平等和合相應的存在者

**五蘊(重心理分析)**

1.  蘊 Skadhan：積聚義(aggregation), 即同類相聚
2.  色：變礙：一切物質
3.  受：領納：有情的情緒作用
4.  想：取像：認識作用, 心即攝取境相而現為心相;由此表象作用, 構成概念, 安立種種名言
5.  行：造作：即意志作用, 主要是”思”心所, 對境而引生內心, 經心思的審慮, 決斷, 出以動身, 發語的行為
6.  識：明了識別, 從能知得名

**五蘊與七十二法**

1.  色(11)：五根, 五境, 五表色
2.  受(1)：受心所
3.  想(1)：想心所
4.  行(58)：

	 - 心相應行(44)
	 - 心不相應行(14)

5.  識(1)：六心王
6.  受,  想別立因其為一切心所中最普遍,  強烈並煩惱相應

**十二處 & 十八界**

1. 內六入(根)：眼耳鼻舌身意
2. 六識：眼識耳識鼻識舌識身識意識
3. 外六處(境)：色聲香味觸法(受,  想,  行)

**諸門分別**

1.  依修行的立場, 由不同的角度去分類或說明諸法的性質, 稱為”諸門分別”或”義門分別”
2.  倫理學：三性(善, 惡, 無記)
3.  修道學：有覆無覆
4.  宗教學：有漏無漏
5.  哲學：有為無為
6.  生理心理融合學：根非根

**倫理學：三性**

1.  行為造作(業)的道德角度判斷可以分為三性：善, 惡, 無記
2.  以好的發心(自利或利他)為三性的根本標準．發心：思(審慮思, 決定思), 思已業(動發思)
3.  善業 —> 可愛異熟, 惡業 —> 非愛異熟
4.  無記 —> 無異熟

**七種善(修行的增進)**

-   善
	 - 有漏
		 - 生得
		 - 加行：聞思修
	 - 無漏
		 - 有為：學(初果, 二果, 三果), 無學(阿羅漢)
		 - 無為：勝義(涅槃)

**四種善(體性)**

1.  勝義善：涅槃, 無為常住的究竟善
2.  自性善：體性本善的有為法, 如三善根(無貪, 無嗔, 無癡)和慚, 愧
3.  相應善：和自性善相應的心, 心所法
4.  等起善：由善心發起的身, 語二業

**P.S. 和以上相反的為四種不善**

**無記**

1.  定義：無善惡性質之法或行為, 無法招感”異熟果”(善惡之果報), 稱為”無記”
2.  無記種類：

	1.  有覆無記：三界之煩惱本身．有覆：能遮蔽聖道和心性
	2.  無覆無記：果報(異熟), 一般行住等行為(威儀路), 工作上專業行為(工巧), 神通心, 自性無記, 勝義無記(兩種無為：虛空無為, 非擇滅無為)

**宗教學：有漏無漏**

1.  漏：比喻法, 以六根洩漏之不淨來比喻不淨的煩惱
2.  有漏：”有”：隨增, 有漏：煩惱隨增, 隨順, 增長煩惱

	1.  相應隨增：心心所法和煩惱相應時增長其勢力
	2.  所緣隨增：所緣隨順著煩惱並增長其力

3.  無漏：不隨增, 不隨煩惱, 苦集二諦是有漏法, 滅道二諦是無漏法
4.  大乘唯識以俱有義解釋有, 就說第七識俱有
